GitHub Pages个人博客搭建

Mar 20, 2019点击

GitHub Pages 间介

对于程序员来说，GitHub的代码托管和开源协作相当便捷，因此它在程序员的心中占有重要的地位。而GitHub提供了一个静态网页的功能，方便使用者进行项目推广和个人博客的搭建，这就是GitHub Pages。

所需工具

  1. gitHub账号（没有的请先注册）

  2. git (可从官网 https://git-scm.com 进行下载，但由于是国外服务器的原因，下载速度巨慢。建议选择自己版本后复制下载链接，使用第三方下载工具进行下载）

  3. html文件（用于展示内容）

创建步骤

  1. 登入自己的GitHub账号，点击加号并选择New repository。

  2. Repository name下输入：‘你的GitHub名字’+‘.github.io’,例如Zkeffort.github.io。
    注：这里一定要用你的GitHub账户名字，这样以后你的博客访问地址就是‘你的GitHub名字.github.io’。当然，你也可以购买一个域名并指向你的GitHub仓库，并通过此域名来访问博客。

    勾选Initialize this repository with a README生成阅读文件，然后点击Create repository进行创建。

  3. 创建完成后你会进入如下画面，这里点击Settings。

  4. 下滑找到GitHub Pages配置，选择master branch分支。

    你会发现如下内容

    Your site is ready to be published at http://zkblog.xyz/ZkeffortDemo.github.io/.

这就是你的博客访问地址。

接下来你就可以通过git这一工具建立与GitHub仓库的连接，并将本地html和其他网页文件上传到GitHub中去，以便更新自己的博客内容。有关git的命令请自行查找，在次不多赘述。

至于博客的主题，推荐大家使用hexo这一框架，支持一键部署，而且具有丰富的主题库和插件库，使用起来非常方便。
